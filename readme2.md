GamesterInc Unity MVP
Private Unity MVP project for GamesterInc — a gamified construction and safety education platform.

🎯 Project Overview
This repository contains an early-stage Unity MVP developed as a proof of concept.

The goal of the product is to teach construction and safety theory through interactive gameplay, not traditional quizzes.

This is not a finished product. It is a foundation that requires refactoring, system design, and scaling.

🎮 Current MVP Features
The current build includes:

Login menu (student + teacher)
Account creation flow
Teacher menu (placeholder, not functional)
Overworld navigation system
NPC interactions
Dungeon-style minigame
Correction-based minigame (safety/PPE)
Character customisation system
Shop interaction
Coin collection
Achievement menu
Pause menu
UI assets and design elements
⚠️ Known Limitations
The MVP is not production-ready and may include:

Hardcoded logic
Incomplete or placeholder systems
Non-functional teacher features
Limited or no backend integration
Quiz-style popups interrupting gameplay
Unstructured or inconsistent code organisation
🧠 Product Vision
GamesterInc is designed to be:

A game where learning happens through decisions and interaction

NOT:

static quizzes
pop-up question screens
Learning should happen through:

Exploring environments
Interacting with NPCs
Identifying hazards
Correcting unsafe behaviour
Selecting tools and actions
Scenario-based gameplay
🔧 Current Development Priority (Milestone 1)
The first priority is to stabilise and understand the existing project.

The developer should:

Open and run the Unity project
Confirm Unity version required
Review project structure
Identify key scenes and scripts
Document what currently works
Identify broken or incomplete features
Highlight hardcoded systems
Recommend a refactor approach
Make only essential fixes for stability
❗ Important Instruction
Do NOT rebuild the project from scratch.

The objective is to:

Preserve useful existing work
Clean and organise the codebase
Make systems reusable and scalable
Prepare for future development
🚀 Future Development Direction
